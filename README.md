🚗 Araç Kiralama Portalı (ASP.NET Core MVC)
Bu proje, bir araç kiralama şirketinin envanterini oluşturmak için geliştirilmiş web tabanlı bir yönetim portalıdır. Yöneticilerin araç eklemesine, listelemesine, Güncellemene ve silmesine (CRUD) olanaklarını tanır.

🛠️ Kullanılan Teknolojiler
Çerçeve: ASP.NET Core 9.0 (MVC)
Veritabanı: Microsoft SQL Server
ORM: Entity Framework Core (Code-First)
Tasarım Kalıbı: Depo Kalıbı ve Genel Depo
Kullanıcı Arayüzü Şablonu: SB Admin 2 (Bootstrap 4 tabanlı)
Kimlik Doğrulama: Çerez Tabanlı Kimlik Doğrulama
Ön uç: Razor Views, HTML5, CSS3, JavaScript, jQuery
✨ Temel Özellikler
Güvenli Giriş: Yönetici girişi için güvenli Çerez tabanlı kimliğin korunması.
Araç Yönetimi: Araçların tablosu, markası, modeli ve günlük kira bedeliyle kayıtlıdır.
Durum Takibi: Araçların düzenlilik düzenlemesinin takibi.
Responsive Tasarım: Tüm cihazlarla uyumlu modern yönetici paneli.
