# 🚗 Araç Kiralama Yönetim Sistemi (ASP.NET Core MVC)

Araç kiralama işletmeleri için geliştirilmiş admin panelli web yönetim sistemidir.

## 🎯 Amaç
Araç, müşteri ve rezervasyon süreçlerini merkezi bir panel üzerinden yönetmek.

---

## 🛠️ Teknolojiler
- ASP.NET Core MVC (.NET 9)
- MSSQL
- Entity Framework Core (Code First)
- ASP.NET Identity
- Bootstrap 4 (SB Admin 2)
- jQuery, JavaScript

---

## ✨ Özellikler
- Araç CRUD işlemleri
- Admin panel yönetimi
- Role-based authorization
- Güvenli giriş sistemi (Cookie Authentication)
- Responsive UI

---

## ⚙️ Mimari
- Repository Pattern
- Katmanlı yapı
